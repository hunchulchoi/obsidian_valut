#### 반수 opposite number
1.	정의: 어떤 수에 더했을 때 덧셈의 항등원인 0이 되는 수를 그 수의 반수라고 합니다.
2.	표현: 실수의 경우, 반수는 원래 수의 절댓값을 유지한 채 부호만 반대로 바꾼 수입니다.
3.	예시:
	•	7의 반수는 -7 (7 + (-7) = 0)
	•	-3.5의 반수는 3.5 ((-3.5) + 3.5 = 0)
	•	0의 반수는 0 (0 + 0 = 0)
4.	성질:
	•	덧셈의 역원(additive inverse)으로도 불립니다.
	•	양수의 반수는 음수이고, 음수의 반수는 양수입니다.
	•	임의의 수 a에 대해, a + (-a) = 0 이 성립합니다.
5.	응용: 뺄셈 연산은 반수를 이용하여 덧셈으로 표현할 수 있습니다. 즉, a - b = a + (-b) 입니다.

#### 절대값 absolute number
표현:
$$
|x|
$$
1.	실수의 경우: 실수 x의 절댓값 |x|는 x가 원점(0)으로부터 떨어진 거리를 나타내는 음이 아닌 실수입니다. 구체적으로:
	•	x > 0 이면 |x| = x
	•	x = 0 이면 |x| = 0
	•	x < 0 이면 |x| = -x
2.	복소수의 경우: 복소수 a + bi의 절댓값은 |a + bi| = √(a² + b²)로 정의되며, 이는 복소평면에서 원점과의 거리를 나타냅니다.
3.	일반적인 정의: 절댓값은 다음 네 가지 조건을 만족하는 함수입니다:
	•	비음수성: |x| ≥ 0
	•	양의 정부호성: |x| = 0 당且 당 x = 0
	•	곱의 법칙: |xy| = |x||y|
	•	삼각부등식: |x + y| ≤ |x| + |y|
#### 소수 prime
- 정의
	- 1보다 큰 자연수
	- 1과 자기 자신만을 약수로 가지는 수
	- -> 합성수
#### 합성수 composite number
- 정의
	- 1과 자신외에 다른 약수를 가진 수
	- -> 소수
#### 최소공배수 LCM, Least Common Multiple
- 두개 이상의 자연수에서 두소의 곱으로 나누어 질수 있는 가장 작은 자연수
- {4, 6}=> 12
- $LCM(a, b) = \frac{|ab|}{gcd(a, b)}$
#### 최대공약수 GCD, Greatest Common Divisor
- 두 수를 나눌때 가장 큰 자연수
- {12, 18} => 6
- $GCD(a, b) = \max\{a, b\} - \min\{a, b\}$
- $GCD(a, b) = \left| a - \frac{a}{b} \cdot b \right|$
#### 단항식 Monomial
- 한개의 변수만을 포함하는 다항식
- e.g)x + 3, 2x - 4
#### 소인수 분해 Prime Factorization
- 어떤 자연수를 다른 자연수의 곱으로 표현하는 것
#### 항등원
- 어떤 집합의 원소와 특정 연산을 수행했을때, 그 원소를 변화 시키지 않는 원소
- 덧셈: 0
- 곱셈: 1
#### 역수 reciprocal
1. 기본 정의:
   - 어떤 수 a의 역수는 1/a입니다 (단, a ≠ 0).
   - a와 그의 역수를 곱하면 항상 1이 됩니다: a * (1/a) = 1
2. 분수의 역수:
   - 분수 a/b의 역수는 b/a입니다 (단, a ≠ 0, b ≠ 0).
   - 분수의 역수는 분자와 분모를 뒤집은 형태입니다.
3. 성질:
   - 0을 제외한 모든 실수는 역수를 가집니다.
   - 1의 역수는 1입니다.
   - 역수의 역수는 원래의 수입니다: (1/a)^-1 = a
4. 응용:
   - 나눗셈은 역수를 이용한 곱셈으로 바꿀 수 있습니다: a ÷ b = a * (1/b)
5. 주의사항:
   - 0은 역수를 가지지 않습니다 (0으로 나눌 수 없기 때문).

#### 역수의 합
- $a+\frac{1}{a} = t$
- $a^2 + \frac{1}{a^{2}} = t^{2}-2$
- $a^3+\frac{1}{a^{3}}=t^3-3t$
#### 결합법칙 (Associative Law)
1. **덧셈에 대한 결합법칙**:
   - 덧셈에서는 괄호 안의 순서가 바뀌어도 결과가 변하지 않습니다.
     $(a + b) + c = a + (b + c)$
2. **곱셈에 대한 결합법칙**:
   - 곱셈에서는 괄호 안의 순서가 바뀌어도 결과가 변하지 않습니다.
     $(a \times b) \times c = a \times (b \times c)$
#### 분배법칙 (Distributive Law)
1. **덧셈에 대한 분배법칙**:
   - 어떤 수를 두 개의 합으로 나타낸 식에 곱할 때, 그 수는 각각의 항에 따로 곱해지고 결과가 더해집니다.
     $a \times (b + c) = a \times b + a \times c$
2. **곱셈에 대한 분배법칙**:
   - 어떤 수를 두 개의 곱으로 나타낸 식에 더할 때, 그 수는 각각의 항에 따로 더해지고 결과가 곱해집니다.
     $a + (b \times c) = (a + b) \times (a + c)$
#### 교환법칙 (Commutative Law)
1. **덧셈에 대한 교환법칙**:
   - 덧셈에서는 순서를 바꿔도 결과가 변하지 않습니다.
     $a + b = b + a$
2. **곱셈에 대한 교환법칙**:
   - 곱셈에서는 순서를 바꿔도 결과가 변하지 않습니다.
     $a \times b = b \times a$
##### 주요 특징
- **결합법칙**: 연산의 괄호 안 순서를 변경할 수 있게 해줍니다.
- **분배법칙**: 덧셈과 곱셈을 재배열하여 계산을 단순화합니다.
- **교환법칙**: 연산의 순서를 변경할 수 있게 해줍니다.
