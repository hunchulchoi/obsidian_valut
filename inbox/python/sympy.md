## 개요
- symbolic python의 약자
- 기호 연산(symbolic math)를 위한 오픈 소스 라이브러리
- 대수(algebra) 문제를 기호 수학으로 풀 수 있다.
	- 방정식의 해구하기
	- 미분과 적분
## 가져오기
`from sympy import *`
## 상수, 변수, 함수, 기호 만들기
- Symbol class를 사용하여 파이썬 객체로서 수학 기호를 다룬다.
- symbol 객체는 수식을 표현하는 데이터 구조의 요소로 사용
